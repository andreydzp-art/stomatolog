# Лендинг «Композитные виниры» — инструкция для Cursor

Одностраничный лендинг на чистом HTML/CSS/JS. Без сборки, без зависимостей —
открывается напрямую в браузере. Шрифты подключаются с Google Fonts.

## Структура

```
composite-veneers/
├── index.html        ← вся страница (разметка + стили в <style> + скрипты в <script>)
└── images/
    ├── doctor-hero2.png   — фото врача в Hero (первый экран)
    ├── doctor-hero.png    — фото врача в блоке «Врач / достижения»
    ├── case-1.webp        — кейс «До/После»
    ├── case-2.webp        — кейс «До/После»
    └── case-3.webp        — кейс «До/После»
```

Весь CSS и JS встроены прямо в `index.html` — отдельных `.css`/`.js` файлов нет.

## Как запустить

Просто откройте `index.html` в браузере. Для удобной разработки в Cursor:
поставьте расширение **Live Server** и нажмите «Go Live».

## Что нужно подключить (главное)

### 1. Ссылки на мессенджеры
По всему файлу кнопки связи помечены атрибутом `data-msg`. Сейчас все ведут на `#`.
Замените `href="#"` на реальные ссылки:

```html
<!-- было -->
<a class="btn ..." href="#" data-msg="telegram">…</a>
<!-- стало -->
<a class="btn ..." href="https://t.me/ВАШ_НИК" data-msg="telegram">…</a>
```

- `data-msg="telegram"` → `https://t.me/username`
- `data-msg="max"`      → ссылка на Max
- `data-msg="instagram"`→ `https://instagram.com/username`

Эти кнопки повторяются: в Hero, в среднем CTA, в финальном блоке, в подвале
и в мобильной нижней панели — замените везде (удобно через «Find & Replace»,
Ctrl/Cmd+Shift+H).

### 2. Имя врача
Найдите `Natalia Kolesnik` и при необходимости замените на нужное имя
(встречается в шапке, блоке достижений и подвале).

### 3. Фото
Замените файлы в `images/` своими (сохранив имена), либо поменяйте `src`.
Рекомендуемые пропорции: Hero — вертикальное 4:5; кейсы — 16:10.

### 4. Тексты и факты
Блок «Врач» (секция `.cred`) содержит достижения и регалии — отредактируйте
заголовки `<h3>` и подписи `<p>` в карточках `.cred-card` под реальные данные.

## Примечания
- Адаптив готов (desktop / tablet / mobile), брейкпоинты в медиа-запросах.
- В самом верху `<body>` есть `<template id="__bundler_thumbnail">` — это превью
  для офлайн-сборки, на саму страницу не влияет. Можно удалить.
- Анимация появления секций — через `IntersectionObserver` (класс `.reveal`).
